# tsdt

A minimal command-line tool that converts UNIX timestamps into human-readable dates.

## Installation

```
pip install tsdt
```

Or for local development using Poetry:

```
poetry install
poetry run tsdt <timestamp>
```

## Usage

Convert a UNIX timestamp:

```
tsdt 1763562191
```

Example output:

```
2025-01-19 14:23:11
```
